How Admin Sync Works

Content added by Admin becomes the default content for Echozy. Existing patients and newly added patients can receive this global content, while caregivers and healthcare providers may add extra personalized content separately.

Content Ownership

Admin content is shared globally. Caregiver or provider content is saved under a selected patient only, so personal phrases will not affect other patients.
