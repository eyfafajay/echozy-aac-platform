# Echozy

Echozy is a web-based multilingual Augmentative and Alternative Communication platform designed for stroke patients with communication impairments.

## Features

- Patient communication board
- Malay and English phrase support
- Text-to-Speech support
- Caregiver phrase customization
- Healthcare provider dashboard
- Patient usage analytics
- Supabase integration
- OpenAI TTS integration

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Supabase
- OpenAI Text-to-Speech API

## Project Purpose

This project was developed as a Final Year Project to support stroke patients who have difficulty speaking by providing a simple, accessible, and multilingual AAC communication platform.
